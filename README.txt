
Kolin Poizon — интерактивный прототип сайта
Содержимое архива:
  - index.html  (полный одностраничный прототип)
  - assets/
      - hero-jacket.jpg
      - p1.jpg ... p6.jpg  (заглушки для карточек)

Как открыть:
  1) Распакуй архив.
  2) Открой файл index.html в браузере (двойной клик).
  3) Для production: замените изображения в /assets на HQ версии, подключите реальный API и Stripe/PayPal.

Нужен React-репозиторий, Figma-файл или бэкенд для платежей? Напиши — сгенерирую.
